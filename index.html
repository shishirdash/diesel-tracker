    Recent weeks

    A minimal-reactivity day = a day with daily tracking where no Reactivity behavior was logged as Issue or Incident. Weeks before you started daily tracking have no per-day data, so they show an est value derived from that week's assessed reactivity severity (Good ≈ 7, Watch ≈ 5, Issue ≈ 2, Incident 0) — a rough heuristic, not a measurement. The headline average counts tracked weeks only.
